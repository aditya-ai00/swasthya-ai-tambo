Swathya AI is an AI-powered health assistant that provides safe, general health guidance.
It does not diagnose medical conditions or replace professional medical advice.

## Features
- Conversational health guidance
- Fever, symptom, and home-care advice
- Clear medical disclaimers
- Emergency escalation guidance
- Built using Tambo AI and Next.js

## Tech Stack
- Next.js (App Router)
- Tambo AI
- TypeScript
- Tailwind CSS

## Safety Disclaimer
Swathya AI provides general health information only.
It does not diagnose conditions or replace a doctor.
If symptoms are severe or persistent, consult a medical professional.
